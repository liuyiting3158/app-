# 【A37】基于智能文字场景个人财务管理创新应用【合合信息】
<a name="rtGmU"></a>
## 1.命题方向
生活服务
<a name="m3XoG"></a>
## 2.题目类别
应用类
<a name="JK3hW"></a>
## 3.题目名称
基于智能文字场景个人财务管理创新应用
<a name="XMplm"></a>
## 4.背景说明
<a name="YssNz"></a>
### 【整体背景】
记账是我国国民的优良传统，记录着生活的支出与收入，有助于居民对生活费用有更清晰了解。市面上大多数的记账APP都需要手工录入才能记账，或获得外卖、支付程序端口才能获取消费信息，存在功能不齐全、隐私过度收集的弊端。此外，中老年人使用记账本应用十分吃力，目前市面上缺乏能够有效调取更全文识别（OCR）技术端口的产品，来将购物小票，购物发票，出租车票，火车票等消费支出凭证里面的消费信息（消费时间，金额，类型等）识别提取出来，录入到应用中，并形成消费行为分析总结。为提升记账APP在个人财务管理中的多场景可用性，给用户以清晰的消费管理目标，需根据不同年龄段使用情况进行产品应用创新，使用户能够高效利用个人财务健康应用，并在产品内容上进行创新，该命题是合合信息目前在新产品创新应用内容方向之一，也是公司重点推进的项目。
<a name="QDNH1"></a>
### 【公司背景】
合合信息是行业领先的人工智能及大数据科技企业，基于自主研发的智能文字识别及商业大数据核心技术，为全球C端用户和多元行业B端客户提供数字化、智能化的产品及服务。<br />企业旗下3款C端APP产品扫描全能王、名片全能王、启信宝服务全球百余个国家和地区的数亿级用户：公司B端服务，即智能文字识别服务、商业大数据服务帮助客户切实解决了降本增效、改善风控、高效获客的业务痛点，提供了较高的技术附加值。<br />核心技术：拥有自主知识产权的113多项发明专利，其中境外专利超过39项；<br />智能文字识别技术:支持超过100种证照识别模块，可识别包含56种语言的文档图像，支持全球不同国家、不同行业的银行卡识别，包含储蓄卡、借记卡、信用卡；
<a name="VG4SS"></a>
### 【业务背景】
合合信息作为拥有行业领先的智能文档处理技术的企业，致力于通过科技创新提高企业与居民的效率与质量。现阶段，智能文档处理技术已经从单一的文字信息提取，发展为可进行信息结构化整合，梳理，分析并给企业用户提供更多决策帮助的“多面手”。个人财务管理对居民生活支出收入做主要记录并创新应用，通过对智能文档处理技术的应用及产品迭代创新，可让居民企业面对日常支出与收入有更清晰了解，为日后生活做好更多规划。
<a name="V8GbF"></a>
## 5.项目说明
<a name="u2f2R"></a>
### 【问题说明】
参赛者将针对青年（19~45岁），中年（45~60岁），老年人（60岁）三大用户群进行产品功能创新。选手将基于不同年龄段进行用户使用调研，通过产品设计，智能文档技术端口调取，获取收入消费支出等内容形成可体验使用的demo 应用（手机 APP，不限于native程序，H5应用等），帮助每个阶段的用户用上高效的个人财务管理工具。
<a name="ePFJc"></a>
### 【用户期望】
通过技术内容创新，能够解决目前用户记账使用各种问题，包括多种消费记录及支出，并形成用户分析行为报告，在AI的智能分析给出消费指导建议。
<a name="KE7eY"></a>
## 6.任务要求
<a name="FLptl"></a>
### 【开发说明】
<a name="NjEOj"></a>
#### （1）用户使用行为分析及调研（必做）20分
①针对不同年龄段使用需求进行功能需求分析及用户用研分析；10分<br />②分析目前市面上的记账本功能存在的功能性差异，并在demo样式上能够提出解决方案及应用方式；10分
<a name="JaWxw"></a>
#### （2）基本功能（必做）60分
**①具备跨设备数据同步功能**<br />A. 用户登录登出功能 5分<br />用户可根据实际情况，进行微信，支付宝等方式进行登录登出功能<br />B. 数据存储到服务器10分<br />在合合信息提供的服务器上进行数据存储<br />**②消费信息录入5分**<br />消费信息包括但不限于：消费时间，金额，消费类型，消费店家。<br />消费类型包括：餐饮，出行，娱乐，学习，日用品及其他。<br />消费凭据种类：包括但不限于外卖小票、购物小票，火车票，出租车发票，电影票，过路费，船运票，客运票等。<br />A.  手动录入5分<br />用户手动输入消费信息。<br />B.  通过对图片 OCR 全文识别，提取信息录入15分<br />支持拍摄或相册选择图片，进行 OCR 全文识别，实现信息提取算法从识别文本中提取识别消费信息。<br />其中消费类型，可以手动选择（消费类型自动判断作为高级功能）。<br />对于自动提取出来的信息，用户可以进行手动修改。<br />饿了么外卖小票，火车票，为必须支持的消费凭证。<br />其他类型，作为可选高级功能。<br />合合信息提供智能文档处理识别接口<br />C. 图像处理增强处理10分<br />对拍摄/导入图片方式录入的，需要对图片进行剪裁增强，弯曲矫正等<br />合合信息提供图像处理接口<br />**③消费数据展示与编辑10分**<br />可以对已录入的消费数据进行修改编，删除操作。<br />使用列表方式，以时间序展示消费数据。可以选择时间段，或消费类型进行搜索过滤。
<a name="pg76Y"></a>
#### （3）高级功能（选做）
**①自动判断消费类型10分**<br />对通过图片 OCR 识别提取的消费信息，进行自动判断消费类型。<br />**②批量图像导入10分**<br />可以从相册中选择多张图像进行消费信息录入。<br />**③多维度展示消费数据10分**<br />必做：<br />两个基础图：<br />A.  折线图，可以按年份，月份筛选，查看某一年或者某一个月的消费趋势<br />B. 饼状图，可以按年份，月份筛选，查看某一年或者某一个月的消费类型占比<br />**④支持消费凭据类型 -每支持一个得5分**<br />支持除饿了么外卖小票，火车票外的其他类型消费凭证。<br />每支持一项加分。<br />**⑤大批量数据测试排名分 5，10，15**<br />我们会提供一份包含 2020 份消费数据的表格，用于模拟大量数据的状态测试。（新账号测试数据加载时长和体验）<br />比赛结束时，参赛队伍需要提供一个包含这些测试数据的账号，用于测试。<br />开发者可以用任意的方式将数据导入到特定账号中，无须在 App 中实现导入测试数据的功能。<br />**⑥个人健康管理功能创新**<br />参赛队伍提出新的功能及想法，有效解决用户的需求（视具体功能点进行评分，具体需要进一步沟通）<br />**⑦单元测试**
<a name="Vcpsn"></a>
### 【技术要求与指标】
①基本功能分 60分<br />参见基本功能项目定义<br />②高级功能分<br />参见高级功能项目定义<br />③ 交互视觉 10分<br />交互易用，顺畅，视觉美观，协调<br />④测试 bug 扣分<br />对实现的功能进行测试，每发现一个 bug，进行扣分<br />Bug 分级：<br />A. 不严重：影响部分功能。扣 2 分<br />B. 严重：crash等，无法使用。扣 4 分<br />⑤设备兼容性 10分<br />要求支持 Android 或 iPhone 设备（形式不限）<br />具体设备的系统版本号要求：支持主流机型系统<br />本次题目考核参赛队伍对于用户使用分析，产品搭建，API调取逻辑及应用结果呈现。
